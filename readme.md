Gravestone
==========

> Stay tuned, more coming!

A budgeting tool piggy backing off [PocketSmith](https://www.pocketsmith.com).

But yeah: 
- TypeScript
- Webpack
- Gulp
- Vue (integrated with TypeScript)
- Dev Server
- Production building

and more coming...

#### Setup
- `npm i webpack gulp typings -g`
- `typings i`
- `npm i`

#### Usage
- `gulp` - dev server
- `gulp build` - production build
